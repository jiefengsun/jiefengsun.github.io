OneDrive Error Message: Sorry, there was problem displaying this folder... please try again

Posted on June 11, 2020 in Computer_Problems

I use OneDrive for Business. The error happens when I changed the directory/folder that I want to synchronize.

The problem is the name of your folder is wrong and the best way is to just let OneDrive degenerate the folders and you move the files that you want to synchronize into tht folder.

To solve the problem follow the following steps:

  1. Try to reinstall OneDrive and reset OneDrive using cmd commend: %localappdata%\Microsoft\OneDrive\onedrive.exe /reset
  2. Name a folder OneDrive exactly.
  3. Select the folder as the folder that you will synchronize.
  4. OneDrive will create a folder named _OneDrive - _, where is your Business domain.
  5. Move all you file to the folder OneDrive - ***.
